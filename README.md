# Bitonic Sort using OpenMP
This repository contains an implementation of the Bitonic Sort algorithm using OpenMP for parallel processing. Bitonic Sort is a parallel sorting algorithm that is particularly well-suited for hardware implementation and parallel computing environments.

# How to RUn

Run in WSL

cd Parallel

gcc -fopenmp -O2 bitonicOmp.c -o bitonicOmp

./bitonicOmp
